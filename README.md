# HistoryDog-privacy
历史汪浏览器扩展的隐私政策页面 / Privacy policy for the HistoryDog browser extension
## 中文说明

这个仓库用于公开托管 `历史汪` 浏览器扩展的隐私政策页面，方便在 Microsoft Edge 插件商店、产品官网或其他公开渠道中引用。

### 仓库用途

- 提供可公开访问的隐私政策页面
- 作为插件商店审核时的隐私政策 URL 来源
- 对外说明插件如何处理浏览器历史记录、用户输入和第三方 AI 服务调用

### 适用产品

- 产品名称：历史汪
- 产品类型：浏览器扩展
- 支持平台：Microsoft Edge / Google Chrome

### 主要说明内容

隐私政策会重点说明以下事项：

- 插件会访问浏览器历史记录，用于本地粗筛候选结果
- 插件会将用户的自然语言查询，以及脱敏后的候选摘要发送到用户选择的 AI 服务商
- 插件不会上传 Cookie、页面正文、query 参数和 hash
- 用户配置的 API Key 保存在本地浏览器扩展存储中，不随扩展安装包分发
- 用户可以修改或删除自己的本地配置与最近搜索记录

## English

This repository hosts the public privacy policy for the `HistoryDog` browser extension. It is intended to provide a stable, publicly accessible URL for the Microsoft Edge Add-ons store, the product website, and other public references.

### Repository Purpose

- Host a publicly accessible privacy policy page
- Provide the privacy policy URL required during extension store submission
- Explain how the extension handles browsing history, user input, and third-party AI service requests

### Product Scope

- Product name: HistoryDog
- Product type: Browser extension
- Supported platforms: Microsoft Edge / Google Chrome

### What the Privacy Policy Covers

The privacy policy should clearly describe:

- The extension accesses browser history to perform local candidate filtering
- The extension sends the user's natural-language query and sanitized candidate summaries to the user-selected AI provider
- The extension does not send cookies, full page content, query parameters, or URL fragments
- User API keys are stored locally in the browser extension storage and are not bundled with the published extension package
- Users can modify or delete their local settings and recent search history
